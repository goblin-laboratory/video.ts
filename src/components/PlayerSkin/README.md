# PlayerSkin

分为直播和回放两种模式

设计参考：
- https://github.com/vidstack/player
- https://vidstack.io/docs/player/components/layouts/default-layout/?styling=default-theme
- https://vidstack.io/player/demo/?layout=default&framework=react&view=player


## LivePlayerBar

- 播放/暂停
- 音量控制：静音，音量调节
- 直播播放时长，时分秒
- 直播：显示一个红色小圆点，后面显示文字“直播”
- 实时、流畅、均衡切换
- 全屏/退出全屏功能


## PlaybackPlayerBar

- 播放进度条，占整行，进度条支持分段，支持预览
- 播放/暂停
- 音量控制：静音，音量调节
- 播放进度/总长度，时分秒
- 全屏/退出全屏功能
